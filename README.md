# 📝 AI Customer Review Summarizer

An AI-powered web application that summarizes customer reviews using both **Extractive** and **Abstractive** Natural Language Processing (NLP) techniques. The application provides an interactive Gradio interface that allows users to compare summarization methods side-by-side.

---

## 🚀 Project Overview

Customer reviews often contain valuable information but can be lengthy and difficult to analyze. This project uses modern NLP techniques to generate concise summaries that help users quickly understand the overall feedback.

Users can:

- Upload or enter customer reviews
- Generate **Extractive Summaries**
- Generate **Abstractive Summaries**
- Compare the results interactively through a Gradio web interface

---

## ✨ Features

- Interactive Gradio web application
- Extractive summarization using Sumy (TextRank)
- Abstractive summarization using the OpenAI API
- Side-by-side comparison of summarization methods
- Real-time summary generation
